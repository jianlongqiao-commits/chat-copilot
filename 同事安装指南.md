# 「把天聊下去」安装指南

直播主持人 AI 副驾——实时听你说话，帮你想下一句该聊什么。

---

## 前提条件

- **Node.js v18+**（下载：https://nodejs.org ，装 LTS 版本就行）
- **Chrome 浏览器**（需要用麦克风，Chrome 兼容性最好）

---

## 需要申请两个 API 密钥

### 1. OpenRouter（AI 追问能力）

1. 打开 https://openrouter.ai ，注册账号
2. 进入 Keys 页面 → **Create Key** → 复制保存
3. 充值几块钱就够用很久（我们用的 DeepSeek 模型非常便宜）

### 2. 火山引擎（语音识别）

1. 打开 https://www.volcengine.com ，注册账号
2. 完成**实名认证**（必须，否则无法开通服务）
3. 进入控制台 → 找到**豆包语音** → 创建应用
4. 勾选「**流式语音识别模型2.0**」
5. 拿到 **APP ID** 和 **Access Token**，复制保存

> 💡 新用户有免费试用包，够测试用。

---

## 安装步骤

### ① 解压 zip

把 `把天聊下去-分发包.zip` 解压到任意目录。

### ② 填写 API 密钥

用文本编辑器打开项目里的 **`.env`** 文件，把三个占位符替换成你自己的 Key：

```
LLM_API_KEY=在这里填入你的OpenRouter API Key
ASR_APP_ID=在这里填入你的火山引擎APP ID
ASR_ACCESS_TOKEN=在这里填入你的火山引擎Access Token
```

其他配置不用动。

### ③ 安装依赖

打开终端（Mac 按 `Cmd+空格` 搜"终端"），运行：

```bash
cd 你的项目目录路径
npm install
```

### ④ 启动

两种方式任选：

- **双击** `启动.command` 文件
- 或者在终端运行 `npm start`

### ⑤ 打开浏览器

访问 **http://localhost:3000** ，搞定。

---

## 使用说明

1. 点击 **「开始转写」** → 浏览器会请求麦克风权限，允许即可
2. 正常说话，页面会实时显示语音转文字
3. **停顿约 2 秒**，AI 会自动生成追问建议；也可以点 **「手动追问」**
4. 可以上传**节目脚本**，AI 会结合脚本内容给出更贴合的建议

---

## 常见问题

**Q：一直显示"连接中"，卡住不动？**
→ 检查 `.env` 里的 ASR 密钥是否正确。确认火山引擎后台已经开通了「流式语音识别模型2.0」。

**Q：LLM 报错 403？**
→ OpenRouter 上部分模型有地区限制。建议使用 `deepseek/deepseek-chat`，没有这个问题。

**Q：没有声音 / 不转写？**
→ 检查 Chrome 是否授权了麦克风权限（地址栏左边的锁图标 → 网站设置 → 麦克风 → 允许）。
